# Humanizer (Aether-Claw bundled skill)

Removes signs of AI-generated writing from text so it sounds more natural and human. Based on [blader/humanizer](https://github.com/blader/humanizer) and Wikipedia’s [Signs of AI writing](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing).

- **Default:** Aether-Claw applies humanizer writing guidelines to all agent replies (see gateway system prompt).
- **Explicit use:** When the user asks to “humanize” or “rewrite to sound more human,” use this skill: read `skills/humanizer/SKILL.md` and follow it to edit the given text.
