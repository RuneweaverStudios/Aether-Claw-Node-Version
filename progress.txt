## Codebase Patterns
- Project is Node.js only (no Python). Use Node 18+.
- Memory files are Markdown-based (brain/*.md). Index with aetherclaw index.
- Config in swarm_config.json; secrets in .env.
- Skills are OpenClaw-style (SKILL.md in skills/). Audit via skill-audit.js.
- Gateway: WebSocket + optional HTTP dashboard in one process (src/daemon.js).
- CLI entry point: src/cli.js (aetherclaw).
